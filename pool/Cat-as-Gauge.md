\boxed{ \mathrm{Law}_{\mathbb P}(Y)=Y_\#\mathbb P = Y_\#\mathbb P_{\mathrm{out}}. } \tag{R.6}

\boxed{ \mathbb P_1\neq \mathbb P_2\ \text{on }\mathcal F \quad\text{but}\quad \mathbb P_1|_{\mathcal G_{\mathrm{out}}}=\mathbb P_2|_{\mathcal G_{\mathrm{out}}}. } \tag{R.7}

\boxed{ Y_\#\mathbb P_1 \approx Y_\#\mathbb P_2 \quad\text{whenever}\quad \mathbb P_1|_{\mathcal G_{\mathrm{out}}}=\mathbb P_2|_{\mathcal G_{\mathrm{out}}}. } \tag{R.8}
这就是“锁定”的强形式：外部推前分布对内部微观差异不敏感。

\boxed{ T_H=\frac{\hbar c^3}{8\pi G M k_B}\ \propto\ \frac{1}{M}. } \tag{R.9}

\boxed{ \text{(无穷远处的 law)} \ \approx\ \text{(温度 }T_H\text{ 的近热 law)}\ \circ\ \text{(灰体滤波 }\Gamma(\omega)\text{)}. } \tag{R.11}
这说明 greybody factor 不是“随手补丁”，而是外部几何散射通道必然诱导的频域过滤。

4) Cat as a Gauge Theory：内部冗余就是 gauge，自整姿态就是 gauge 控制

把系统变量分成两类：

base：外部关心的整体姿态（比如躯干姿态 R 或其坐标）
shape：内部关心的形状变量（关节角、脊柱弯曲、分段相对旋转等）
猫的“魔法”在于：它能通过改变 shape（内部形状与惯量分布）来改变 base（整体姿态），而不需要外部施加同等规模的净扳矩。
4.2 外部等价类：gauge 的严谨定义

定义关系：两条轨迹 ω1, ω2 如果 Y(ω1) = Y(ω2)，则对外部观察者完全不可区分。
于是，一整个等价类（很多内部轨迹）对应同一个外部结果。内部轨迹的自由选择空间，就是本节所称的 gauge 自由。

这就是“猫问题”的核心：内部回路产生外部姿态净变化。你可以把“回路的选择”理解为“gauge 的选择”。

5) 带宽与过滤：Big-M + Nyquist 二次形（为什么高频细节会被抹掉）

粗糙度二次形（Nyquist 代理）大意是：把相邻控制指令的差分平方求和，再乘上 M^2 做尺度对齐。直觉上：

M 越大，可允许的变化越快；

7) 四脚落地极限（condition on 一只具体猫）

扰动项：随 ε 增大而变差（wind、初始误差、噪声等）
带宽项：随 M 增大而变好（更细的时间分辨率、更可实现的括号/shape 控制）

1 - p_{ε,M}(θ) ≤ C1 * r(ε) + C2 / M
