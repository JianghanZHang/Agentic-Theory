## Critical Findings (requires action)

### C1. Acquisition Sign Error — argmax UCB for Cost Minimization

**Severity:** LOAD-BEARING + UNRESOLVED

**Passage** (optimal-cycle.tex:361-370):
```
\beta_n \gets \begin{cases}
  \beta^{(n)} & n \leq 2d+1 \\
  \arg\max_{\beta \in S_n}[\mu_{n-1}(\beta) + \sqrt{\bar\beta_n}\,\sigma_{n-1}(\beta)]
  & n > 2d+1
\end{cases}
```

Also at optimal-cycle.tex:273 (rmk:omd-regret): `selects β_n = argmax u_n(β)`.

**Problem:** The algorithm maximizes the upper confidence bound μ + √β̄σ, but J is a **cost** to minimize:
- Return statement (line 392): `argmin_n J_n`
- Regret definition (line 277): `R_T = Σ[J(β_n) - J(β*)]` where β* minimizes J

For cost minimization, GP-LCB selects `argmin[μ - √β̄σ]`, not `argmax[μ + √β̄σ]`. As written, the algorithm seeks the point with the highest plausible cost.

**Propagation:** The safe set definitions are also inconsistent:
- S₁ (line 356): UCB ≤ c_max (μ + √β̄σ ≤ c_max) — pessimistic, correct for safety
- S_{n+1} (line 387): LCB ≤ c_max (μ - √β̄σ ≤ c_max) — optimistic, expansion criterion
- Proof of prop:safe-gp (line 438): says "line 7 uses the lower confidence bound" and claims safety w.h.p. — but LCB ≤ c_max does NOT guarantee safety

**Impact:** All six guarantee propositions cite Algorithm 2. The regret bound (prop:gp-regret) invokes Srinivas GP-UCB [Theorem 1], which is for reward maximization, not cost minimization.

**Resolution options:** Either (a) negate J: define reward r = -J, model r with GP, acquisition becomes correct; or (b) switch to GP-LCB: `argmin[μ - √β̄σ]` with regret defined as minimization regret.

### C3. "Sigma Points as Minimal Generators" Overclaims Sufficiency

**Severity:** LOAD-BEARING + CONTRADICTED (by rmk:mpc)

**Passage** (optimal-cycle.tex:837-851, rmk:sigma-algebra):
```
For Gaussian measures, the σ-algebra generated by 2d+1 sigma-point
evaluations captures all second-order information:
  F_{2d+1}^{UT} = σ(J(β^(1)),...,J(β^(2d+1))),
  E[J(β)|F_{2d+1}^{UT}] = μ_{GP}(β),
The sigma points are the *minimal generators* of a sufficient
σ-algebra for the Gaussian prior.
```

**Three sub-problems:**

**(i) Sufficiency contradicts Lyapunov decrease.** rmk:mpc (line 592) states V_n = tr(Σ_n) is **strictly decreasing** with each observation. If 2d+1 sigma points were truly sufficient, then V_{2d+2} = V_{2d+1} — no further improvement, contradicting strict decrease.

**(ii) The equation in rmk:unscented Level 3 (lines 754-757) includes sigma-point weights w^{(j)}.** The standard GP posterior is μ(β) = k(β,X)(K + σ²I)⁻¹J with no sigma-point weights. Including w^{(j)} makes the formula incorrect unless w^{(j)} = 1 for all j (which is not the standard UT weighting).

**(iii) "Minimal generators" is misleading.** A σ-algebra generated by 2d+1 observations is strictly coarser than one with 2d+2 observations. Calling sigma points "sufficient" implies no further observations improve the posterior, which is false for GPs with SE kernel.

**What is actually true:** The unscented transform with 2d+1 points exactly captures the mean and covariance of a d-dimensional Gaussian pushed through a nonlinear function. This is a finite-dimensional moment-matching result, not a function-space sufficiency result.

### S1. Independence Claim Between G and F_n Is False

**Passage** (optimal-cycle.tex:877-878):
```
G is independent of F_n — the injected noise does not leak
into the acquisition decisions.
```

**Problem:** F_n = σ(β₁,J₁,...,β_n,J_n). Each J_n = (1/M)Σ J^(m)(β_n) depends on θ^(m) ∈ G. Therefore J_n ∈ σ(G), so F_n ∩ G ≠ {∅,Ω}.

**Correct statement:** G_n (noise at step n) is independent of F_{n-1} (acquisition was determined before noise draw). The acquisition β_{n+1} is F_n-measurable and determined before G_{n+1} is realized. But J_n carries the noise into F_n, so G and F_n are not independent.

**Contradicted by** rem:domain-rand (line 701): "The domain-randomized cost contributes observation noise σ²_noise to the GP likelihood, which the posterior update absorbs naturally" — i.e., the noise IS in F_n.
